# Case Assets

Diese Dateien sind neutrale CAD-Platzhalter fuer anonymisierte Industrie-Cases.

Sobald echte Kundenfreigaben vorliegen, Dateien mit denselben Namen ersetzen:
- `ersatzteil-cad.svg`
- `ersatzteil-cad-detail.svg`
- `ersatzteil-cad-einbau.svg`
- `montagehilfe-cad.svg`
- `montagehilfe-cad-detail.svg`
- `montagehilfe-cad-arbeitsplatz.svg`
- `prototyp-cad.svg`
- `prototyp-cad-variante-a.svg`
- `prototyp-cad-variante-b.svg`

Empfehlung:
- Format: `.webp` oder `.jpg`
- Breite: mindestens 1600 px
- Keine sensiblen Kundendaten oder Zeichnungsnummern sichtbar lassen
