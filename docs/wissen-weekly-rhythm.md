# Wissen Weekly Rhythm (0-50 EUR Budget)

## Ziel
- Jede Woche ein belastbarer Fachbeitrag fuer industrielle B2B-Anfragen.
- Jeder Beitrag fuehrt klar in den Funnel (`/projekt-starten/`).

## Wochenablauf (60-90 Minuten)
1. Thema aus realen Leads waehlen (haeufigste Rueckfrage).
2. Beitrag schreiben:
   - Problem im Industriealltag
   - Technische Einordnung
   - Konkrete Handlungsempfehlung
   - Naechster Schritt: `Projekt starten`
3. Beitrag auf `/wissen/` veroeffentlichen.
4. Beitrag als LinkedIn-Post mit Link verteilen.

## Redaktionsregeln
- Kein Clickbait, kein Hobby-Fokus.
- Klarer Industriebezug (Maschinenbau, Produktion, Instandhaltung).
- Keine unrealistischen Expressversprechen.
- CTA immer identisch:
  - Primary: `Projekt starten`
  - Optional Secondary: `Rueckruf anfragen`

## 8 Themenvorschlaege
1. Ersatzteil-Nachfertigung bei Maschinenstillstand
2. Materialwahl fuer temperaturkritische Bauteile
3. Toleranzen im FDM-Industrieeinsatz
4. Kleinserie ohne Werkzeug: Kosten- und Zeithebel
5. Montagehilfen fuer Prozesssicherheit
6. Risikoanalyse bei funktionskritischen Teilen
7. Lieferfenster realistisch planen statt Express-Irrtum
8. Checkliste fuer technische Projektanfragen
